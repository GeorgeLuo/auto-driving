Milestone 006 Pre-Plan: Decision-Facing Perception Readiness

After evidence memory exists, decide whether the packaged physical perception evidence is good enough to inform a first constrained non-idle decision path—or promote exactly one bounded upgrade under the same perception contract. This is immediate deferred work from physical parity, not an open-ended CV backlog.

Status: pre-plan - queued after 005 Drafted: 2026-07-18 Prerequisite: milestone 005 Action policy: still idle until exit gates Algorithm budget: at most one promotion attempt Delivery: rolling PR horizon when activated

Why This Pre-Plan Exists

Milestone 004 proved always-on observation, exact latest publication, and operator inspection on the Pi while retaining lightweight_observer / floor-plane as the operational control. Lab floor_continuity was rejected with reject_keep_control. Residual physical limits remain: distance-sensitive side misses and clear-floor false positives.

That path is sufficient for milestone 005 memory (retain attributed evidence with visual provenance). It is not yet an honest basis for movement authority. Architecture does not need a rethink; evidence quality for decision use may. This pre-plan freezes that question as the next milestone after memory, without turning residual issues into an indefinitely expanding research list.

Pre-plans are not active work. Do not implement 006 while 005 is open unless an explicit scope decision says otherwise. Revise this document only when new closeout evidence changes the bounded question.

High-Level Objective

Measure Decision Fitness

Score packaged floor-boundary evidence against decision-facing claims (presence, zone, clear floor, survival across brief dropout) using labeled physical and controlled sequences—not aesthetic overlays alone.

One Bounded Upgrade Path

If the control is insufficient, qualify at most one candidate or package change under the existing perception contract. Explicit reject-and-keep-control remains a valid outcome.

Stop Before Open Search

Do not start a multi-candidate tournament, semantic recognition program, or metric mapping project. Close with a written sufficiency decision for the next action milestone.

Success is a durable answer: either “packaged evidence is fit for a named constrained decision experiment under idle-to-shadow gates,” or “control retained; the single upgrade attempt failed; later work must restate a new adoption gate.” Success is not a perfect detector.

Scope Boundaries

In Scope

  • Decision-facing fitness measures on the packaged control.
  • At most one promotion attempt under the existing perception contract.
  • Common-frame qualification, optional Pi viability if promoted.
  • Written sufficiency decision for the next action-oriented milestone.
  • Use of 005 memory for temporal stability scoring when available.

Out Of Scope

  • Open multi-candidate tournaments or “try everything in lab/”.
  • Semantic recognition, VLM products, metric SLAM, or floor-plan reconstruction.
  • Rewriting the perception stage contract or always-on publication path.
  • Non-idle movement authority or collision-avoidance claims.
  • Absorbing every residual from 002–004 into one milestone.
